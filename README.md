# ParkMCR - Real-Time Parking Web App

## Overview
ParkMCR is a full-stack web application that simulates real-time parking availability using a Node.js backend and an interactive map frontend.

## Features
- Real-time parking simulation
- Interactive map (Leaflet.js)
- REST API backend (Node.js / Express)

## Technologies Used
- Node.js
- Express.js
- JavaScript (Frontend)
- Leaflet.js

## How to Run

1. Install dependencies:
npm install

2. Start server:
node server.js

3. Open browser:
http://localhost:3000

## Notes
This is a prototype system using simulated data. No real database is implemented.
